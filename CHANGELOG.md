# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增
- 添加锁定护甲功能，一键锁定单位护甲为 2E+20
- 添加 GitHub Actions 自动构建和发布工作流
- 添加自动打包脚本，构建后自动创建 Release 文件夹

### 更改
- 将目标框架从 .NET Framework 4.6 升级到 4.8
- 改进构建脚本，添加自动打包功能
- 更新 README 和 BUILD_GUIDE 文档

### 修复
- 修复构建脚本中的路径问题
- 改进错误处理机制

## [1.0.0] - 2024-12-02

### 新增
- 初始发布
- 支持修改游戏资源（金、木、人口等）
- 支持修改单位属性（HP、MP、攻击力、护甲等）
- 支持修改英雄属性（力量、敏捷、智力、经验值等）
- 支持多个游戏版本（1.20e 到 1.28f）

### 技术细节
- 基于 .NET Framework 4.8
- Windows Forms 界面
- 内存读写功能
